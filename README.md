# portfolio

minimal, brutalist portfolio.

## stack

- next.js (app router)
- tailwind css
- typescript

### installation

```bash
# clone
git clone https://github.com/parthesh28/portfolio.git
cd portfolio

# install
npm install
```

### development

```bash
npm run dev
```

visit `http://localhost:3000`

### production

```bash
npm run build
npm start
```
